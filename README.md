# Smallfeed

**Smallfeed**는 소규모 팀이나 커뮤니티를 위한 **공동 큐레이션 피드** 서비스입니다.  
- 멤버끼리 발견한 블로그 글, 유튜브 영상을 비공개 피드에 함께 모아놓을 수 있습니다.
- 콘텐츠를 읽을 때는 **개인화된 추천**과 **자동 요약**을 받을 수 있습니다.





## 기능 소개

- **초대 기반의 공유 공간** - 관심사를 공유하는 피드 멤버끼리 함꼐 발견한 컨텐츠를 모아놓을 수 있습니다.

- **개인화 추천** - 피드에 올라오는 추천 순서는 나만의 취향을 반영합니다.

- **자동 요약** - 글, 유튜브 영상을 등록하면 자동으로 내용을 요약해서 미리보기로 표시해 줍니다.

- **하이브리드 검색** - 키워드 검색과 벡터 유사도 검색을 모두 지원합니다. 

- **간단한 초대, 회원가입** - 초대 링크만 공유하면 아직 회원가입 하지 않은 사용자라도 링크를 눌러 바로 피드에 참여할 수 있습니다.

## 프로젝트 배경

- 사내 스터디를 진행하던 도중 느낀 불편함을 해결하려고 만든 서비스입니다.
<details>
  <summary>자세히</summary>  


- **문제:**
  - 스터디 멤버끼리 기술 트렌드, 서비스 사례 등을 서로 공유하고 싶었는데 마음에 쏙 드는 플랫폼이 없었습니다.
  - **단톡방**은 시간이 지나면 빠르게 시야에서 사라지니 공유된 컨텐츠를 바로 읽지 않으면 놓쳐버리기 쉬웠습니다.
  - **Notion**이나 **구글 독스**는 정리할 때는 좋지만 가볍게 컨텐츠를 열람하기에는 불편했습니다.
  - 함꼐 스터디를 하고 있어도 개인별로 **세부 관심사가 조금씩 다른데** (e.g. 프론트, 백엔드, 인프라..) 각 개인의 취향에 쏙 맞는 내용부터 먼저 보고 싶었습니다.

- **해결책:**
  - 검색, 요약, 추천을 제공하는 무한스크롤 피드형 서비스를 만들어 직접 사용하기로 했습니다.

</details>
